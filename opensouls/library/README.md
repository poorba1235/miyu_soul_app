# Library

This is a copy paste library for the soul engine.

## Install

Any of these files can be easily installed:

```bash
bunx soul-engine install <pathToFile>
```

for example:

```bash
bunx soul-engine install cognitiveSteps/externalDialog
```

You can also install multiple files at once:

```bash
bunx soul-engine install cognitiveSteps/externalDialog cognitiveSteps/internalMonologue subprocesses/summarizeConversation
```
